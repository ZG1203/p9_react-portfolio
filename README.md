# Week9 React Portfolio

## Description
A React-based portfolio displays Github repository of eight previous weekly challenges. 

## Installation
1. Clone the code locally
2. Install dependencies
    bash
    npm install
3. Start the server 
    bash
    npm run dev
4. Access the application
    Open the provided local server address in your browser to view your app.

## Technologies Used
- React
- Axios for API calls
- GitHub REST API
